AutoResize JS

Make iframe automatically adjust height according to the contents without using scrollbar

Installation

Add this line to the shop

    
        <script type="text/javascript" src="//sales.spreadshirt.net/_resources/autoResize/shop.js"></script>

Add this snippet to your website

    
        <iframe src="//611779.spreadshirt.net" id="shop" width="100%"></iframe>

        <script>
            AUTORESIZE.init(
                document.getElementById('shop')
            );
        </script>
